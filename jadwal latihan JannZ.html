Jadwal dengan Jumat sebagai hari istirahat

by JannZ

Senin - Back, Traps, Bicep

  • Pull-up neutral grip (3 set failure)
  • Lat pulldown wide grip (3 set failure)
  • Single arm cable row (3 set failure)
  • Barbel/cable row (3 set failure)
  • Dumbbell shrug (3 set failure)
  • Bicep curl beban berat (3 set failure) + 1 set beban ringan (failure)
  • Hammer curl (3 set failure)

Selasa - Shoulder, Tricep

  • Superset dumbbell shoulder press > front raise (10 reps x 3 set)
  • Superset lateral raise beban berat > beban ringan (3 set failure)
  • Upright row (3 set failure)
  • Dumbbell rear delt (10–12 reps x 3 set)
  • Reverse pec dec beban berat + 1 set beban ringan (failure)
  • Tricep pushdown beban berat + 1 set beban ringan (failure)
  • Reverse tricep pushdown beban berat + 1 set beban ringan (failure)

Rabu - Chest, Bicep, Forearm

  • Incline bench press (8–10 reps x 3 set)
  • Chest press (3 set failure)
  • Chest fly beban berat + 1 set beban ringan (failure)
  • Superset incline dumbbell curl beban berat + 1 set beban ringan (failure)
  • Hammer curl (3 set failure)
  • Superset cable wrist curl, reverse cable wrist curl, reverse ez bar curl (3 set failure)

Kamis - Back, Traps, Tricep

  • Pull-up neutral grip (3 set failure)
  • Lat pulldown wide grip (3 set failure)
  • Single arm cable row (3 set failure)
  • Barbel/cable row (3 set failure)
  • Dumbbell shrug (3 set failure)
  • Tricep pushdown beban berat + 1 set beban ringan (failure)
  • Reverse tricep pushdown beban berat + 1 set beban ringan (failure)

Jumat - Rest Day

Hari istirahat untuk pemulihan otot. Fokus pada nutrisi, tidur cukup, dan hidrasi yang baik.

Sabtu - Leg

  • Superset walking lunges > Romanian deadlift > calf raises (8–10 reps x 3 set)
  • Bulgarian split squat (8–10 reps x 3 set)
  • Squat (8–10 reps x 3 set)
  • Leg extension (3 set failure)
  • Leg press (10–12 reps x 3 set)

Minggu - Chest, Shoulder

  • Incline bench press (8–10 reps x 3 set)
  • Chest press (10 reps x 3 set)
  • Chest fly (3 set failure)
  • Dumbbell shoulder press (10 reps x 3 set)
  • Lateral raise (3 set failure)
  • Reverse pec dec (3 set failure)
